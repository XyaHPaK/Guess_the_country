# Guess-the-country
This is an educational game that will increase your knowledge of geography.

## Get Started
- clone repository
- 'npm install'
- 'npm install -g parcel'
- 'parcel ./src/index.html'
- Open [http://localhost:1234](http://localhost:1234) in your browser.
